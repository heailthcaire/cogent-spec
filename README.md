# cogent-spec
A proposed prompt-language convention for composable, testable LLM skills.

Cogent is a thin convention layer for structuring LLM skills and AI workflows. It adds imports, methods, namespaces, return contracts, control flow, error handling, versioning, and runtime postconditions to plain-language prompting.

The goal is not to replace natural language. The goal is to make AI workflows easier to compose, inspect, test, reuse, and improve.
